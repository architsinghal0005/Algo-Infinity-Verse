Introduction to Two Pointers

The Two Pointers technique uses two indices that move through an array, string, or linked list to solve problems efficiently. Instead of using nested loops, we intelligently move pointers to reduce time complexity from O(N²) to O(N).

Why Two Pointers?

Two pointers help reduce nested loop solutions into a single traversal. By moving pointers intelligently, many O(N²) problems can be solved in O(N).

💡

Whenever a problem involves sorted arrays, palindrome checking, pair finding, or linked lists, consider using the Two Pointers technique.

02

Left Right Pointer Technique

One pointer starts from the beginning and another from the end. They move toward each other based on a condition.

C++ 

                        vector<int> twoSum(vector<int>& nums, int target)
                        {
                            int left = 0;
                            int right = nums.size() - 1;

                            while(left < right)
                            {
                                int sum = nums[left] + nums[right];

                                if(sum == target)
                                    return {left, right};

                                if(sum < target)
                                    left++;
                                else
                                    right--;
                            }

                            return {};
                        }

Common Uses

  • Two Sum in Sorted Array
  • Container With Most Water
  • Palindrome Validation
  • Removing Duplicates
03

Fast Slow Pointer Technique

Fast and Slow pointers move at different speeds. This technique is heavily used in linked list problems.

C++ 

                    bool hasCycle(ListNode* head)
                    {
                        ListNode* slow = head;
                        ListNode* fast = head;

                        while(fast && fast->next)
                        {
                            slow = slow->next;
                            fast = fast->next->next;

                            if(slow == fast)
                                return true;
                        }

                        return false;
                    }
Example 
                        1 → 2 → 3 → 4
                            ↑     ↓
                            ← ← ←

                        Slow and Fast pointers eventually meet.
                        Cycle Detected = True

Applications

  • Cycle Detection
  • Finding Middle Node
  • Happy Number
  • Detect Loop Start

Applications of Two Pointers

The Two Pointers technique is widely used in arrays, strings, and linked lists. By moving pointers intelligently, many brute-force O(N²) solutions can be optimized to O(N).

05

Practice Problems

Test your understanding of the Two Pointers.

Exercise 1: Two Sum II

Given a sorted array [2,7,11,15] and target = 9, find the indices of the pair.

Answer: [0,1]

Explanation: Using the Two Pointers technique, place one pointer at the beginning (2) and another at the end (15). Since 2 + 15 = 17 is greater than 9, move the right pointer left. Continue until 2 + 7 = 9, which matches the target. Therefore, the required indices are [0,1].

Exercise 2: Valid Palindrome

Is "racecar" a palindrome?

Answer: Yes

Explanation: Compare characters from both ends of the string using two pointers. The first and last characters are both 'r', then 'a' and 'a', followed by 'c' and 'c'. All corresponding characters match, and the pointers meet at the center character 'e'. Therefore, "racecar" is a palindrome.

Exercise 3: Middle of Linked List

Which technique is used to find the middle node of a linked list in one traversal?

Answer: Fast and Slow Pointers

Move the slow pointer one step and the fast pointer two steps. When the fast pointer reaches the end, the slow pointer will be at the middle node of the linked list.
